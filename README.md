# Cert-Easy
### 本脚本仅支持DNS验证申请证书
### 1. 安装
```
sudo bash -c 'wget -O /usr/local/bin/cert-easy https://raw.githubusercontent.com/Lanlan13-14/Cert-Easy/refs/heads/main/acme.sh && chmod +x /usr/local/bin/cert-easy && cert-easy'
```
### 2. 已安装？执行
```
sudo cert-easy
```
### 3. 卸载
##### 卸载选项在脚本中已提供

- 请先前往凭据管理添加相关信息后再申请证书
